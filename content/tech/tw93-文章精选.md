## 你不知道的 AI Coding：非技术人的上手、场景与实战

### 原文链接 `<https://x.com/HiTw93/status/2048230976447557787>`

### 非技术同学怎么把 Claude Code 真正用起来，而不是停在对话框 AI

1. 终端并不是程序员专属，关键是先跨过第一步
2. 非技术同学也需要懂一点框架、Git、报错和代码结构
3. Claude Code 更像通用 Agent，写代码只是最早的入口
4. 最适合交给 AI 的，是目标清楚、结果好验收的任务
5. 从 software for one 开始，先做只服务自己的小工具
6. 需求要写具体：问题、范围、异常、验收标准都要落到数字和场景
7. 复杂任务先用 Plan Mode，确认方案后再让它执行
8. 改代码前用 Git 留检查点，排错时先找根因再动手
9. CLAUDE.md、Skill、Memory 是把个人习惯沉淀成工作流的关键
10. 截图、拆小任务、重启对话和安全边界，是新手最容易忽略的基本功
